ChattyApp
=====================

A minimalist optionally anonymous chat app using React.  

### Dependences

```
babel-core
babel-loader
babel-preset-es2015
babel-preset-react
css-loader
node-sass
sass-loader
sockjs-client
style-loader
webpack
webpack-dev-server
react
react-dom
express
ws
uuid
```

### Usage

To start the server:

```
npm i
npm start
open http://localhost:3000
```
